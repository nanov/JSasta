# Debug Symbols Feature Demo

## Compilation with -g flag:
./build/jsastac -g examples/test_break_continue.jsa test_debug.ll

## Compile to executable:
clang -g test_debug.ll -o test_debug

## Debug with lldb:
lldb test_debug

## Features implemented:
✓ -g flag added to compiler
✓ LLVM DIBuilder integration
✓ Compile unit with file information
✓ Debug info for main function
✓ Line-by-line debug locations for all statements
✓ Proper DWARF version metadata (version 4)
✓ Debug Info Version metadata (version 3)
✓ Works with lldb debugger
✓ Source code display in debugger
✓ Breakpoints by line number
